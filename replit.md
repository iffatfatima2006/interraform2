# Project Overview

This is a React + Vite + TypeScript web application bootstrapped as a fresh project.

## Tech Stack

- **Frontend**: React 19 with TypeScript
- **Build Tool**: Vite 8
- **Package Manager**: npm

## Project Structure

- `src/` - React source code
  - `App.tsx` - Main application component
  - `main.tsx` - Entry point
  - `App.css` / `index.css` - Styles
- `public/` - Static assets
- `index.html` - HTML entry point
- `vite.config.ts` - Vite configuration (port 5000, host 0.0.0.0, allowedHosts: true)

## Development

The app runs on port 5000 via the "Start application" workflow using `npm run dev`.

## Deployment

Configured for Replit deployment.
